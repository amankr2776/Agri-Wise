# KisanMitra - National Agricultural Grid

KisanMitra is a professional, AI-powered platform designed to provide a unified intelligence and logistics grid for the Indian agricultural ecosystem.

## 🚀 Presentation Highlights
*   **Role-Based Command Centers:** Specialized interfaces for Farmers, Agri-Scientists, and Logistics Providers.
*   **AI Crop Diagnostics:** Multimodal disease identification using Google Gemini and Genkit.
*   **Mandi-Link Pro:** A real-time logistics marketplace with fare intelligence and net-profit estimators.
*   **Bio-Security Surveillance:** Geospatial threat monitoring for Experts and Authorities.
*   **Regional Accessibility:** Deep support for 15+ Indian languages with integrated Voice Assistance.

## 🛠 Tech Stack
*   **Framework:** Next.js 15 (App Router)
*   **UI/UX:** Tailwind CSS, ShadCN UI, Framer Motion, Recharts
*   **Backend:** Firebase (Auth, Firestore, Hosting)
*   **AI Engine:** Google Genkit + Gemini 2.5 Flash
*   **State:** Zustand (with Persist)
